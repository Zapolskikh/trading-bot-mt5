RiskManager: Инициализация RiskManager из конфига #1
Описание: парсинг risk-конфига, валидация параметров, инициализация инвариантов.

RiskManager: Проверка возможности открытия сделки #2
Описание: оценка дневного остатка риска, проверка лимита активных сделок, валидация дистанции стопа и pip value.

RiskManager: Расчёт размера позиции #3
Описание: формула лотов по equity, % риска, стоп-дистанции и pip value, нормировка по шагу/минимуму лота.

RiskManager: Учёт состояний сделок (register_new/fill/close) #4
Описание: обновление дневного использованного риска, управление списком активных сделок, освобождение риска при закрытии.

RiskManager: Динамический риск (опционально) #5
Описание: правила адаптации % риска при дроудауне/серии убытков.

MetaTraderClient: Подключение к терминалу #6
Описание: connect/disconnect, обработка ошибок, ретраи, таймауты.

MetaTraderClient: Получение рыночных данных #7
Описание: get_tick/get_market_data для разных таймфреймов и окна, нормализованные DataFrame.

MetaTraderClient: Торговые операции #8
Описание: place/modify/close, идемпотентность заявок, возврат идентификаторов и статусов.

MetaTraderClient: Портфель и символы #9
Описание: get_portfolio (balance/equity/margin) и get_symbol_info (digits/point/lot_step/min_lot/contract_size/tick_value).

Strategy: Подготовка данных #10
Описание: сбор и нормализация баров по символам/таймфреймам, настройка окна/barsize/period.

Strategy: Индикаторы #11
Описание: расчёт и добавление EMA/ATR и др. индикаторов в DataFrame по конфигу.

Strategy: Entry/Exit #12
Описание: генерация сигналов входа/выхода с параметрами side, sl, tp, confidence.

Strategy: Real-time мониторинг #13
Описание: формирование статуса стратегии, последних сигналов и значений индикаторов.

Strategy: Расчёт пипов #14
Описание: корректный расчёт пипов для символов с разной точностью (например, JPY пары).

AlertService: Настройка и отправка уведомлений #15
Описание: отправка сигналов, обновлений ордеров, риск-алертов и ошибок в Telegram; обработка сбоев.

JournalService: Схемы CSV и запись событий #16
Описание: ведение файлов signals.csv, orders.csv, trades.csv; ротация, запись заголовков, консистентность.

TradeEngine: Главный цикл и оркестрация #17
Описание: получение данных, запуск стратегии, риск-проверки, исполнение через клиент, логирование и алерты.

TradeEngine: State machine заявок #18
Описание: переходы NEW→PLACED→FILLED→CLOSED, обработка REJECTED/EXPIRED/CANCELLED, отражение в журналах.

TradeEngine: Ежедневные процедуры #19
Описание: reset_daily_limits, ротация CSV, health-check и восстановление после сбоев.
